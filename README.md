# QA Data Validation Project

Projeto prático de **Quality Assurance** focado em validação de dados, testes de API e automação de testes.

O objetivo deste projeto é simular um fluxo completo de QA utilizando dados públicos de municípios brasileiros.

## 🎯 Objetivos do Projeto

Este projeto demonstra práticas comuns utilizadas por profissionais de QA em sistemas corporativos:

- Validação de integridade de dados
- Análise de qualidade de dados
- Testes de API
- Automação de testes

## 🛠 Tecnologias Utilizadas

- PostgreSQL
- SQL
- Node.js
- Express
- Postman
- Playwright

## 📊 Dataset

O projeto utiliza dados públicos de municípios brasileiros contendo:

- CÓDIGO DO MUNICÍPIO - TOM
- CÓDIGO DO MUNICÍPIO - IBGE
- MUNICÍPIO - TOM
- MUNICÍPIO - IBGE
- UF

Esses dados são utilizados para realizar análises de qualidade e validações.
